# Две кучи (Two Heaps)

## Описание

Паттерн **Две кучи** использует две кучи (min-heap и max-heap) для эффективного
решения задач, связанных с поиском медианы, поддержанием отсортированного порядка
или разделением данных на две части.

## Когда использовать

- Нужно найти медиану потока чисел
- Задачи на разделение данных на две части
- Поиск k-го наибольшего/наименьшего элемента
- Задачи на приоритеты

## Признаки задачи

- "Найти медиану потока чисел"
- "Разделить массив на две части с минимальной разницей сумм"
- "Найти медиану в потоке данных"
- "K-й наибольший элемент в потоке"

## Как это работает

1. Min-heap хранит большую половину элементов (минимальный элемент сверху)
2. Max-heap хранит меньшую половину элементов (максимальный элемент сверху)
3. Медиана находится на вершине одной из куч или как среднее двух вершин

## Сложность

- **Временная сложность**: O(log n) для вставки, O(1) для получения медианы
- **Пространственная сложность**: O(n)

## Типичные задачи

1. Медиана потока чисел
2. Разделение массива на две части
3. Sliding Window Median
4. Find Median from Data Stream
