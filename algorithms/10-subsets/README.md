# Подмножества (Subsets)

## Описание

Паттерн **Подмножества** используется для генерации всех возможных подмножеств
множества. Это фундаментальная задача комбинаторики, решаемая с помощью рекурсии
и backtracking.

## Когда использовать

- Генерация всех подмножеств
- Генерация всех перестановок
- Генерация всех комбинаций
- Задачи на поиск всех возможных вариантов

## Признаки задачи

- "Найти все подмножества массива"
- "Найти все перестановки"
- "Найти все комбинации размера k"
- "Генерация всех возможных вариантов"

## Подходы

### 1. Рекурсивный подход
Используем рекурсию для построения подмножеств, добавляя или не добавляя каждый элемент.

### 2. Итеративный подход
Начинаем с пустого множества и постепенно добавляем элементы.

### 3. Битовая маска
Используем битовые маски для представления подмножеств.

## Сложность

- **Временная сложность**: O(2^n) — количество подмножеств экспоненциально
- **Пространственная сложность**: O(2^n) для хранения результата

## Типичные задачи

1. Все подмножества
2. Подмножества с дубликатами
3. Перестановки
4. Комбинации размера k
5. Генерация скобок
