# Два указателя (Two Pointers)

## Описание

Паттерн **Два указателя** использует два указателя для итерации по структуре данных
одновременно. Указатели могут двигаться в одном направлении или навстречу друг другу.

## Когда использовать

- Работа с отсортированными массивами или связанными списками
- Поиск пар элементов, удовлетворяющих условию
- Сравнение элементов с двух концов структуры
- Удаление дубликатов на месте

## Признаки задачи

- "Найти пару чисел с заданной суммой в отсортированном массиве"
- "Удалить дубликаты из отсортированного массива"
- "Проверить, является ли строка палиндромом"
- "Найти три числа с суммой, равной нулю"

## Варианты использования

### 1. Указатели с разных концов
Один указатель в начале, другой в конце. Движутся навстречу друг другу.

### 2. Указатели в одном направлении
Оба указателя начинают с одного конца, но движутся с разной скоростью.

## Сложность

- **Временная сложность**: O(n) — каждый элемент обрабатывается максимум один раз
- **Пространственная сложность**: O(1) — используются только два указателя

## Типичные задачи

1. Пара с целевой суммой (отсортированный массив)
2. Удаление дубликатов
3. Три числа с суммой ноль (3Sum)
4. Возведение в квадрат отсортированного массива
5. Подмассив с произведением меньше целевого
