# Слияние интервалов (Merge Intervals)

## Описание

Паттерн **Слияние интервалов** эффективно работает с перекрывающимися интервалами.
Часто требуется найти пересечения, объединения или вставить новый интервал.

## Когда использовать

- Задача связана с интервалами (отрезками времени, диапазонами)
- Нужно найти перекрывающиеся интервалы
- Нужно объединить интервалы
- Нужно вставить интервал в отсортированный список

## Признаки задачи

- "Объединить перекрывающиеся интервалы"
- "Вставить интервал и объединить перекрытия"
- "Найти пересечение двух списков интервалов"
- "Проверить, можно ли посетить все встречи"

## Определение перекрытия

Два интервала [a, b] и [c, d] перекрываются, если:
- a <= d И c <= b

Или проще: начало одного интервала <= конец другого (для обоих).

## Шесть случаев взаимного расположения интервалов

1. `a` полностью до `b`: [a_start, a_end] ... [b_start, b_end]
2. `a` и `b` перекрываются, `a` начинается раньше
3. `a` полностью внутри `b`
4. `a` и `b` перекрываются, `b` начинается раньше
5. `b` полностью внутри `a`
6. `b` полностью до `a`

## Сложность

- **Временная сложность**: O(n log n) — обычно требуется сортировка
- **Пространственная сложность**: O(n) — для хранения результата

## Типичные задачи

1. Слияние интервалов
2. Вставка интервала
3. Пересечение интервалов
4. Проверка конфликтов встреч
5. Минимальное количество переговорных комнат
