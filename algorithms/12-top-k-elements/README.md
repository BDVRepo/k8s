# Топ K-элементов (Top K Elements)

## Описание

Паттерн **Топ K-элементов** используется для нахождения K наибольших или наименьших
элементов в коллекции. Обычно решается с помощью кучи (heap) или быстрой сортировки.

## Когда использовать

- Найти K наибольших/наименьших элементов
- Найти K наиболее частых элементов
- Найти K ближайших точек
- Задачи на приоритеты

## Признаки задачи

- "Найти K наибольших элементов"
- "Найти K наиболее частых чисел"
- "Найти K ближайших точек к началу координат"
- "K-й наибольший элемент"

## Подходы

### 1. Min-Heap для K наибольших
Используем min-heap размера K, чтобы хранить K наибольших элементов.

### 2. Max-Heap для K наименьших
Используем max-heap размера K, чтобы хранить K наименьших элементов.

### 3. Quick Select
Используем модифицированную быструю сортировку для поиска K-го элемента.

## Сложность

- **Временная сложность**: O(n log k) с кучей, O(n) в среднем с Quick Select
- **Пространственная сложность**: O(k) для кучи

## Типичные задачи

1. K наибольших элементов
2. K наиболее частых элементов
3. K ближайших точек
4. K-й наибольший элемент в массиве
5. Топ K частых слов
