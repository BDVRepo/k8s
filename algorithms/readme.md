# Алгоритмы: 14 паттернов для решения задач на собеседованиях

Этот репозиторий содержит изучение и реализацию 14 основных паттернов программирования,
которые помогут решить большинство алгоритмических задач на технических собеседованиях.

## Содержание

| # | Паттерн | Папка | Когда использовать |
|---|---------|-------|-------------------|
| 1 | [Скользящее окно](./01-sliding-window/) | `01-sliding-window` | Подмассивы/подстроки заданного размера K |
| 2 | [Два указателя](./02-two-pointers/) | `02-two-pointers` | Отсортированные массивы, поиск пар |
| 3 | [Быстрые и медленные указатели](./03-fast-slow-pointers/) | `03-fast-slow-pointers` | Обнаружение цикла, поиск середины |

| 5 | [Циклическая сортировка](./05-cyclic-sort/) | `05-cyclic-sort` | Массивы с диапазоном [0, n] |
| 6 | [Разворот связанного списка](./06-reverse-linked-list/) | `06-reverse-linked-list` | Разворот на месте |

| 12 | [Топ K-элементов](./12-top-k-elements/) | `12-top-k-elements` | K наибольших/наименьших |

## Структура каждой папки

```
XX-pattern-name/
├── README.md      # Описание паттерна, когда использовать, анализ сложности
├── main.go        # Реализация с подробными комментариями
└── main_test.go   # Тестовые примеры
```

## Как использовать

1. Изучите README.md каждого паттерна для понимания концепции
2. Прочитайте код с комментариями в main.go
3. Запустите тесты: `go test -v`
4. Попробуйте решить похожие задачи самостоятельно
