# Модифицированный бинарный поиск (Modified Binary Search)

## Описание

Паттерн **Модифицированный бинарный поиск** расширяет классический бинарный поиск
для работы с отсортированными массивами, которые могут быть ротированы, или для
поиска границ, вставки элементов и других модификаций.

## Когда использовать

- Массив отсортирован, но может быть ротирован
- Нужно найти границы диапазона
- Поиск в бесконечном отсортированном массиве
- Поиск элемента в 2D матрице
- Поиск пика в массиве

## Признаки задачи

- "Найти элемент в ротированном отсортированном массиве"
- "Найти первую/последнюю позицию элемента"
- "Найти пик в массиве"
- "Поиск в 2D отсортированной матрице"

## Варианты модификаций

### 1. Ротированный массив
Массив отсортирован, но повёрнут вокруг точки.

### 2. Поиск границ
Найти первую или последнюю позицию элемента.

### 3. Поиск пика
Найти локальный максимум в массиве.

### 4. Бесконечный массив
Поиск в массиве неизвестного размера.

## Сложность

- **Временная сложность**: O(log n) — как и классический бинарный поиск
- **Пространственная сложность**: O(1)

## Типичные задачи

1. Поиск в ротированном отсортированном массиве
2. Поиск диапазона элемента
3. Поиск пика
4. Поиск в бесконечном массиве
5. Поиск в 2D матрице
