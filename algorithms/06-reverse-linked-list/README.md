# Разворот связанного списка (Reverse Linked List)

## Описание

Паттерн **Разворот связанного списка** используется для обращения порядка элементов
в связанном списке. Это фундаментальная операция, которая часто встречается в задачах.

## Когда использовать

- Нужно изменить порядок элементов в связанном списке
- Задачи на палиндромы в связанных списках
- Разворот части списка
- Разворот по группам

## Признаки задачи

- "Развернуть связанный список"
- "Развернуть связанный список между позициями m и n"
- "Развернуть связанный список по группам k"
- "Проверить, является ли список палиндромом"

## Варианты разворота

### 1. Полный разворот
Разворачиваем весь список.

### 2. Частичный разворот
Разворачиваем часть списка между двумя позициями.

### 3. Разворот по группам
Разворачиваем каждые k элементов.

## Сложность

- **Временная сложность**: O(n) — проходим список один раз
- **Пространственная сложность**: O(1) для итеративного подхода, O(n) для рекурсивного

## Типичные задачи

1. Развернуть связанный список
2. Развернуть связанный список между позициями
3. Развернуть по группам k
4. Палиндром в связанном списке
5. Развернуть узлы в четных позициях
