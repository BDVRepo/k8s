# Дерево DFS (Depth-First Search)

## Описание

Паттерн **DFS (обход в глубину)** используется для обхода дерева, двигаясь как можно
глубже по каждой ветке перед возвратом. Есть три основных варианта: Pre-order,
In-order и Post-order.

## Когда использовать

- Задачи на пути в дереве
- Поиск элементов в дереве
- Вычисление сумм, максимумов и т.д.
- Построение деревьев
- Задачи на симметрию

## Признаки задачи

- "Найти путь с максимальной суммой"
- "Проверить, существует ли путь с заданной суммой"
- "Найти все пути от корня до листьев"
- "Проверить симметрию дерева"

## Варианты обхода

### 1. Pre-order (Корень → Левое → Правое)
Сначала обрабатываем корень, затем левое поддерево, затем правое.

### 2. In-order (Левое → Корень → Правое)
Сначала левое поддерево, затем корень, затем правое.

### 3. Post-order (Левое → Правое → Корень)
Сначала левое поддерево, затем правое, затем корень.

## Сложность

- **Временная сложность**: O(n) — посещаем каждый узел один раз
- **Пространственная сложность**: O(h), где h — высота дерева (для стека рекурсии)

## Типичные задачи

1. Поиск пути с заданной суммой
2. Все пути от корня до листьев
3. Сумма всех путей
4. Проверка симметрии дерева
5. Диаметр дерева
