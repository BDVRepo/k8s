# Топологическая сортировка (Topological Sort)

## Описание

Паттерн **Топологическая сортировка** используется для упорядочивания вершин
направленного ациклического графа (DAG) таким образом, что для каждого рёбра
(u, v) вершина u идёт перед v в упорядочении.

## Когда использовать

- Граф не имеет направленных циклов (DAG)
- Нужно обновить все объекты в отсортированном порядке
- Есть класс объектов, которые следуют определённому порядку
- Задачи на зависимости и упорядочивание

## Признаки задачи

- "Планирование задач с зависимостями"
- "Найти порядок выполнения задач"
- "Минимальная высота дерева"
- "Курсы с предварительными требованиями"

## Алгоритм

1. Вычисляем входящие степени (in-degree) для каждой вершины
2. Добавляем все вершины с in-degree = 0 в очередь
3. Извлекаем вершину из очереди и добавляем в результат
4. Уменьшаем in-degree всех соседей на 1
5. Если in-degree соседа становится 0, добавляем его в очередь
6. Повторяем, пока очередь не пуста

## Сложность

- **Временная сложность**: O(V + E), где V — количество вершин, E — количество рёбер
- **Пространственная сложность**: O(V + E)
