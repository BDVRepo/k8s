# Скользящее окно (Sliding Window)

## Описание

Паттерн **Скользящее окно** используется для выполнения операций над определённым
"окном" (подмассивом или подстрокой) входных данных. Окно "скользит" по массиву,
добавляя элементы справа и удаляя слева.

## Когда использовать

- Входные данные — линейная структура (массив, связанный список, строка)
- Нужно найти подмассив/подстроку, удовлетворяющую условию
- Задача связана с непрерывной последовательностью элементов

## Признаки задачи

- "Найти максимальную сумму подмассива размера K"
- "Найти наименьший подмассив с суммой больше S"
- "Найти самую длинную подстроку с K различными символами"

## Виды скользящего окна

### 1. Фиксированное окно
Размер окна задан заранее (например, K элементов).

### 2. Динамическое окно
Размер окна меняется в зависимости от условий задачи.

## Сложность

- **Временная сложность**: O(n) — каждый элемент обрабатывается максимум дважды
- **Пространственная сложность**: O(1) для простых случаев, O(k) если нужна хеш-таблица

## Типичные задачи

1. Максимальная сумма подмассива размера K
2. Наименьший подмассив с суммой ≥ S
3. Самая длинная подстрока с K различными символами
4. Строка с перестановкой паттерна
5. Наименьшее окно, содержащее подстроку
