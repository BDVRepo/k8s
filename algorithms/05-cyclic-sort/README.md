# Циклическая сортировка (Cyclic Sort)

## Описание

Паттерн **Циклическая сортировка** используется для работы с массивами, содержащими
числа в определённом диапазоне. Вместо сравнения элементов, мы используем тот факт,
что каждое число должно находиться на своей позиции (индекс = значение).

## Когда использовать

- Массив содержит числа в диапазоне [0, n] или [1, n]
- Нужно найти пропущенные, дублирующиеся или неправильно расположенные числа
- Задача связана с индексами массива

## Признаки задачи

- "Найти все пропущенные числа в массиве [1, n]"
- "Найти дубликаты в массиве [1, n]"
- "Найти первое пропущенное положительное число"
- "Найти все числа, исчезнувшие из массива"

## Как это работает

1. Итерируемся по массиву
2. Для каждого элемента проверяем, находится ли он на правильной позиции
3. Если нет, меняем его с элементом на правильной позиции
4. Повторяем, пока текущий элемент не окажется на правильной позиции

## Сложность

- **Временная сложность**: O(n) — каждый элемент обрабатывается максимум один раз
- **Пространственная сложность**: O(1) — сортировка на месте

## Типичные задачи

1. Найти все пропущенные числа
2. Найти дубликат
3. Найти все дубликаты
4. Найти первое пропущенное положительное число
5. Найти все исчезнувшие числа
