# Дерево BFS (Breadth-First Search)

## Описание

Паттерн **BFS (обход в ширину)** используется для обхода дерева уровень за уровнем.
Начинаем с корня, затем обрабатываем все узлы на уровне 1, затем на уровне 2 и так далее.

## Когда использовать

- Обход дерева по уровням
- Поиск кратчайшего пути в дереве
- Задачи на уровни дерева
- Поиск элементов на определённом уровне

## Признаки задачи

- "Обойти дерево по уровням"
- "Найти минимальную глубину дерева"
- "Найти элементы на уровне k"
- "Обойти дерево зигзагом"

## Как это работает

1. Используем очередь для хранения узлов текущего уровня
2. Обрабатываем все узлы текущего уровня
3. Добавляем дочерние узлы в очередь для следующего уровня
4. Повторяем, пока очередь не пуста

## Сложность

- **Временная сложность**: O(n) — посещаем каждый узел один раз
- **Пространственная сложность**: O(w), где w — максимальная ширина дерева

## Типичные задачи

1. Обход дерева по уровням
2. Минимальная глубина дерева
3. Уровень с максимальной суммой
4. Зигзагообразный обход
5. Связывание узлов на одном уровне
