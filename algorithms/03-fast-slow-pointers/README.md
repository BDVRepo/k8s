# Быстрые и медленные указатели (Fast & Slow Pointers)

## Описание

Паттерн **Быстрые и медленные указатели** (также известный как "Заяц и Черепаха")
использует два указателя, которые движутся с разной скоростью по структуре данных.
Обычно медленный указатель движется на один шаг, а быстрый — на два.

## Когда использовать

- Работа с циклическими связанными списками или массивами
- Поиск середины связанного списка
- Определение, является ли связанный список палиндромом
- Поиск начала цикла в связанном списке

## Признаки задачи

- "Определить, есть ли цикл в связанном списке"
- "Найти середину связанного списка"
- "Найти начало цикла"
- "Проверить, является ли число счастливым"

## Как это работает

1. Медленный указатель движется на 1 шаг за итерацию
2. Быстрый указатель движется на 2 шага за итерацию
3. Если есть цикл, указатели обязательно встретятся
4. Если цикла нет, быстрый указатель достигнет конца

## Сложность

- **Временная сложность**: O(n) — в худшем случае проходим весь список
- **Пространственная сложность**: O(1) — только два указателя

## Типичные задачи

1. Обнаружение цикла в связанном списке
2. Начало цикла в связанном списке
3. Счастливое число (Happy Number)
4. Середина связанного списка
5. Палиндром в связанном списке
