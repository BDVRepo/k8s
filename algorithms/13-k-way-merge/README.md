# K-way слияние (K-way Merge)

## Описание

Паттерн **K-way слияние** используется для объединения K отсортированных списков
в один отсортированный список. Обычно решается с помощью кучи (min-heap).

## Когда использовать

- Объединение K отсортированных списков
- Объединение K отсортированных массивов
- Слияние отсортированных интервалов
- Задачи на объединение отсортированных данных

## Признаки задачи

- "Объединить K отсортированных списков"
- "Объединить K отсортированных массивов"
- "Найти K-й наименьший элемент в K отсортированных списках"
- "Слияние отсортированных интервалов"

## Как это работает

1. Создаём min-heap и добавляем первые элементы из каждого списка
2. Извлекаем минимум из кучи и добавляем в результат
3. Добавляем следующий элемент из того же списка в кучу
4. Повторяем, пока куча не пуста

## Сложность

- **Временная сложность**: O(n log k), где n — общее количество элементов, k — количество списков
- **Пространственная сложность**: O(k) для кучи

## Типичные задачи

1. Объединение K отсортированных списков
2. K-й наименьший элемент в K отсортированных списках
3. Объединение K отсортированных массивов
4. Слияние отсортированных интервалов
